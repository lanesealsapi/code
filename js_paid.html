Variables

Variables store data values. Use let for changeable values and const for constants.

let name = "L.A.x";
const pi = 3.14;

Functions

Functions let you group reusable code blocks into a single call.

function greet(person) {
  return "Hello " + person + "!";
}

Conditionals

Conditionals let your code make decisions based on conditions.

let age = 18;
if (age >= 18) {
  "Adult";
} else {
  "Minor";
}

Loops

Loops repeat tasks without rewriting the same code.

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Objects

Objects group data and behavior into key-value pairs.

let person = {
  name: "L.A.x",
  age: 25
};

Arrays

Arrays store ordered lists of values.

let colors = ["red", "green", "blue"];

Events

Events let your page react to user interactions.

document.querySelector("button")
  .addEventListener("click", () => {
    alert("Clicked!");
  });

DOM

The DOM is how JavaScript interacts with web pages.

document.getElementById("demo").textContent = "Hi!";
