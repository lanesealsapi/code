Conditionals

Use if, elif, and else for decisions.

age = 18
if age >= 18:
    print("Adult")
else:
    print("Minor")

Loops

Use for to iterate, while for repetition.

for i in range(1, 6):
    print(i)

Lists

Lists store ordered collections of items.

colors = ["red", "green", "blue"]
print(colors[0])

Dictionaries

Dictionaries map keys to values.

person = {"name": "L.A.x", "age": 25}
print(person["name"])

Classes

Classes bundle data and behavior together.

class Dog:
    def __init__(self, name):
        self.name = name

spark = Dog("Spark")
print(spark.name)

Modules

Import modules to reuse code from the Python standard library.

import math
print(math.sqrt(16))
